#Global covid tracker 
## General info
This project is simple mini covid tracker with fatelity rate and visualization.
	
## Technologies
Project is created with:
* Python 3.6
* Django 3.1
* API : covid19api
	
## Setup
To run this project, one should have django and python installed.
